Who created this site?

This site was created by an independent web developer as a fun project.

Why was this site created?

I created this site to help those looking to get and edge when taking their written test.

How does it work?

After the test is taken you're given either a "Passing score" or "Failing score". These scores are based off of the real California written test and reflects how the actual test will score you.

Why is this site the best for you?

I designed this site to complete free and a great source to test your ability to pass the California Written Test.
